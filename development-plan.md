**作成予定のディレクトリ構造:**
```

├── components/          # UIコンポーネント
│   ├── dashboard/      # ダッシュボード関連
│   ├── modal/          # モーダル関連
│   ├── common/         # 共通コンポーネント
│   └── ui/             # v0生成コンポーネント
├── hooks/              # カスタムフック
├── utils/              # ユーティリティ関数

├── data/               # 初期データ
└── styles/             # スタイル関連
public/
└── assets/             # Terrariaアセット
    └── icons/          # アイテムアイコン
```

## 📝 開発時の注意事項

### **コーディング規則**
- CLAUDE.md の開発ガイドラインを厳守
- TypeScript を使用した型安全な開発
- コンポーネント指向の設計
- レスポンシブデザインの考慮

### **データ管理**
- ローカルストレージのキー名: `terraria-collection-status`
- 公式アセットのみ使用（自作アイコン禁止）
- JSON形式でのデータ管理

### **UI/UX原則**
- シンプルで直感的な操作
- 所持アイテム: カラー表示
- 未所持アイテム: グレーアウト
- 明確な視覚フィードバック
